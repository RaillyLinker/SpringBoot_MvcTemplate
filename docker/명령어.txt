[도커 설명]
springboot 서버 프로그램 실행

[Dockerfile]
(이미지 빌드)
> docker image build --pull=true -f Dockerfile-Server-Dev -t raillylinker/SpringBoot_MvcTemplate:latest .
or
> docker image build --pull=true -f Dockerfile-Server-Prod -t raillylinker/SpringBoot_MvcTemplate:latest .

(컨테이너 중단)
> docker container stop SpringBoot_MvcTemplate

(컨테이너 파기)
> docker container rm SpringBoot_MvcTemplate

(컨테이너 생성 및 실행)
> docker container run -d -e TZ=Asia/Seoul --rm -p8083:8083 --name SpringBoot_MvcTemplate raillylinker/SpringBoot_MvcTemplate:latest
or
> docker container run -d -e TZ=Asia/Seoul --rm -p80:80 --name SpringBoot_MvcTemplate raillylinker/SpringBoot_MvcTemplate:latest

(사용하지 않는 모든 이미지 삭제)
> docker image prune -af

(컨테이너 명령창 띄우기)
> docker container exec -it SpringBoot_MvcTemplate sh
